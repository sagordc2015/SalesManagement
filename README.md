# SalesManagement
Android mobile application for the sole trader ship business. Including user, company profile, suppliers, products, purchases, customers and sales.

Necessary Components:
=====================
SQLite
SharedPreferences
MPAndroidChart
Custom Adapter
Custom Search
Gallery Access & Save to Directory
Custom Alert Dialog
Validation
Exception Handling
MVC Design Pattern
